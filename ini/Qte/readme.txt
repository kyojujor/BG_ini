Qte按键
KeyMode:按键模式  [0:为引导模式 1:多键值判定模式,默认为0]
在[0按键模式] 和 [1按键模式]下需要不同方式处理的字段

Name:            按键中文包名称, 可配置数量不限,适用模式[0按键模式] 或 [1按键模式]
Value:           键值,适用模式[0按键模式] 或 [1按键模式] 
NotifyImage:     提示动画名称 适用模式[0按键模式] 或 [1按键模式]
AlignOffsetXY:   多键值模式相对偏移量, 适用模式[1按键模式]
SucceedEvent:    成功执行的行为  适用模式[0按键模式] 或 [1按键模式]
